@EJB annotation

You can use the @EJB annotation to refer a session bean in other components specified by the Java EE platform, for instance:

  • Servlets
  • ServletContextListeners
  • Servlet Filters
  • JSF managed beans
  • EJB interceptors
  • JAX-WS service endpoints
