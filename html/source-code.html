Browsing

You can browse the source via the web interface to Subversion.

Downloading

Or via a Subversion client you can download (checkout) the source with the following URL http://svn.apache.org/repos/asf/incubator/openejb/trunk/openejb3

Checking that out from the command line via subversion is as easy as:

$ svn co http://svn.apache.org/repos/asf/incubator/openejb/trunk/openejb3

Building

To build the code, you'll need to grab a copy of Maven version 2.0.4 or higher. The standard build command is as follows:

$ cd openejb3
$ mvn clean install -Dassemble=true

The output of that command should end with "BUILD SUCCESSFUL"
